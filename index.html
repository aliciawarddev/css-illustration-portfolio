CSS Illustration Gallery

by Alicia Ward

A collection of illustrations created entirely with CSS

About This Project

Each illustration in this gallery is created using only HTML and CSS - no images, SVGs, or canvas elements. This project explores the artistic possibilities of CSS, pushing the boundaries of what can be achieved with code alone.

Techniques used include gradients, transforms, pseudo-elements, clip-paths, animations, and creative use of borders and shadows. Click on any piece to see the full illustration and examine the code behind it.
